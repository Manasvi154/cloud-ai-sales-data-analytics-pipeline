3.1 Create bucket

2. Bucket name -> DATA_BUCKET
3. Region -> AWS_REGION
4. Keep "Block all public access" ON

   code/artifacts/sagemaker/

   reports/generated/

3. Rule 2: processed/ + feature-engineered/ -> transition after 30 days
4. Rule 3: reports/generated/ -> expire after 180 days

16) TROUBLESHOOTING FAST PATH
============================================================
Issue: Upload works but Step Function never starts
Check:
1. S3 trigger prefix exactly raw/uploads/
2. Lambda env STATE_MACHINE_ARN set correctly
3. Lambda role has states:StartExecution

Issue: Step Function fails at Glue
Check:
1. Glue job name in ASL matches actual job
2. Glue role has read/write bucket access
3. Glue script path valid

Issue: SageMaker training fails
Check:
1. training_code_s3_uri points to train_model.tar.gz
2. image URI valid for region
3. sagemaker role has S3 read/write

Issue: Insights empty
Check:
1. BEDROCK_MODEL_ID set and access granted
2. Lambda role includes bedrock:InvokeModel / bedrock:Converse
3. fallback insights still appear even if model call fails

Issue: Dashboard iframe blank
Check:
1. quicksight lambda env values are correct
2. allowed domain includes http://127.0.0.1:5000
3. quicksight user arn/dataset/dashboard id are correct
